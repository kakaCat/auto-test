## 使用规范（与用户指南的边界）

- 用户指南仅描述“行为层事件契约”（事件名、触发时机、载荷字段、错误约定），不内嵌组件的 Props/Emits/Expose 表。
- 组件接口细节统一在本目录维护，并提供最小示例代码，供用户指南通过链接引用。
- 若组件跨页面复用，建议单独成文并在相关用户指南建立双向引用。
