## 🚀 快速开始

1. **环境准备**
   ```bash
   npm install
   npm run dev
   ```

2. **开发规范**
   - 阅读 [编码风格指南](./guides/coding-style.md)
   - 了解 [API架构](./api/README.md)

3. **项目结构**
   ```
   src/
   ├── api/          # API接口层
   ├── components/   # 通用组件
   ├── views/        # 页面组件
   ├── stores/       # 状态管理
   └── utils/        # 工具函数
   ```
