## 文档组织原则

- **文件夹分类**: 使用文件夹对不同类型内容进行隔离
- **职责单一**: 每个文件夹都有明确的职责边界
- **层次清晰**: 采用合理的目录层次结构
- **易于维护**: 便于查找、更新和扩展

## 技术栈

- **框架**: Vue 3 + Composition API
- **语言**: TypeScript
- **构建**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router
- **UI库**: Element Plus / 自定义组件
